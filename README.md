# Task Manager
Modern Todo App built with React and Tailwind CSS.

## Live Demo   https://1task-manger.netlify.app/

## Features

- Add Tasks
- Delete Tasks
- Complete Tasks
- Dark Mode
- Local Storage
- Toast Notifications
- Responsive Design
- Framer Motion Animations

## Technologies

- React.js
- Tailwind CSS
- Framer Motion
- React Hot Toast

npm install
